# unity timeline 타임라인

## 패키지 설치
* 에디터의 메뉴 드롭다운에서 Window > Package Manager를 선택합니다.
* Package Manager 창에서 Advanced > Show Preview Packages를 선택 (package가 release되어 더이상 pre-release 옵션을 켜지 않아도 됩니다.)
