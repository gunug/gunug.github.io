# Gemini 제미나이 실습2

# 스크립트 생성하기
* ```새 스크립트 만들어줘. clickEventHandler.cs gameObject를 마우스로 클릭 할때마다 OnMouseClickCustom이라는 이벤트를 발생시키고 inspector에서 다른 컴퍼넌트 함수를 연결해서 쓸수 있도록.```
* ```새 스크립트 colorChanger.cs파일 생성해줘. material color를 빨강,파랑,초록 순서대로 반복되게 변경하는 public 함수 colorRotation 포함```

* Hirarchy 창 마우스 우클릭 Create > Cube로 큐브 생성

* Cube를 선택하고 inspector창에 새로 생성된 clickEventHandler.cs와 colorChange.cs 파일을 드래그 드랍하여 등록

<img style='border:solid 1px black;' src="https://image.onethelab.com/resized/1762184763.jpg" />

* 추가(+) 버튼 클릭 후 Object 란에 colorChanger드래그 하여 등록하면 Function 부분에서 public 함수를 선택할수 있음
* colorRotation 선택하면 OnClickCustom(이벤트)와 colorRotation(기능)을 연결 할 수 있음.
* 결과 적으로 '클릭(OnclickCustom)되면 컬러바꾸기(colorRotation)' 기능이 동작하도록 됨.
